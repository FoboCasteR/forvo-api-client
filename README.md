# ForvoApiClient

[Forvo API](http://api.forvo.com/) client

## Installation

Add this line to your application's Gemfile:

    gem 'forvo_api_client'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install forvo_api_client
